# **App Name**: WealthWatch

## Core Features:

- Portfolio Snapshot: Displays a high-level overview of all investment holdings.
- Manual Asset Entry: Allows users to manually input investment holdings details.
- Appraisal Projection: Generates an assessment of holdings based on current information, and alerts users if details about their assets needs clarification from a tool.
- Holdings Summary: Show current market value, cost basis, and total gain/loss for each holding.
- Historical Performance: Visualizes portfolio performance over different time periods with interactive charts.
- Asset Allocation: Illustrates how assets are distributed across different investment types.
- User Authentication: Provides secure user registration and login functionality.

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E) to convey trust and stability.
- Background color: Light gray (#F5F5F5) for a clean and professional look.
- Accent color: Muted teal (#4DB6AC) to highlight key data and actions.
- Body and headline font: 'Inter', a sans-serif font providing a modern, objective feel. This font will be used for both headings and body copy.
- Use simple, geometric icons to represent different asset classes and portfolio metrics.
- Implement a grid-based layout for a structured and data-rich presentation.
- Incorporate subtle transitions and animations to enhance user engagement.
